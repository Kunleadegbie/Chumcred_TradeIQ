# 📈 TradeIQ™ — NGX Portfolio Intelligence Platform

## 🚀 Overview

TradeIQ™ is an AI-powered investment intelligence platform designed specifically for the Nigerian Stock Exchange (NGX).

It helps investors, analysts, and institutions make smarter decisions by combining:

- Portfolio Optimization
- Risk Monitoring
- Stock Analysis
- AI-Driven Insights

---

## 🎯 Problem We Solve

Unlike global markets, NGX lacks:

- Structured, accessible data
- Analytical tools for retail investors
- Decision-support systems for stockbrokers

TradeIQ bridges this gap.

---

## 💡 Key Features

### 📊 Dashboard
- Market snapshot
- Top performers & laggards
- Portfolio insights

### 📈 Market Overview
- Performance comparison
- Correlation heatmaps
- Volatility analysis

### 🔍 Stock Analyzer
- Individual stock breakdown
- Trend analysis (MA20 / MA50)
- Risk & performance scoring

### 💼 Portfolio Builder
- Build portfolios by strategy:
  - Low Risk
  - Balanced
  - Growth
  - Dividend Focus

### ⚖️ Portfolio Optimizer
- Optimize using:
  - Sharpe Ratio
  - Sortino Ratio
- Download allocations

### 📉 Risk Monitor
- Value at Risk (VaR)
- Drawdown analysis
- Diversification score
- Correlation risk

### 🧠 AI Insights
- Plain-English explanations
- Market interpretation
- Portfolio commentary
